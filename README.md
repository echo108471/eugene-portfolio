# eugene-portfolio
Personal portfolio for Eugene Cho, built with Next.js, React, and Tailwind CSS.

## Design Direction

The site should feel like a personal engineering portfolio, not a SaaS landing page. The design target is quiet, direct, and recruiter-readable: Eugene first, work second, impact in context.

See [design.md](./design.md) for the full design direction.

## Tech Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
